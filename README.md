# Student Marks Predictor

## Overview
This project predicts student marks using a regression-based machine learning model.

## Features
- Train regression model using dataset
- Predict student marks based on input
- Model evaluation

## Technologies Used
- Python
- Pandas
- NumPy
- Scikit-learn

## Dataset Features
study_hours
attendance
previous_marks
sleep_hours
internet_usage
assignment_score
participation
final_marks

1. Clone the repository
